# 🏠 House Price Prediction using Machine Learning

A Machine Learning web application that predicts house prices based on various house features. The project is built using Python, Scikit-learn, and Streamlit.

---

## 📌 Project Overview

This application predicts house prices using a trained **Random Forest Regressor** model. Users can enter house details through a simple web interface and instantly receive a predicted house price.

---

## 🚀 Features

- 📊 Interactive Dashboard
- 🏠 House Price Prediction
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning Model Information
- 👨‍💻 About Project Page
- 💰 Price Prediction in Indian Currency Format
- 🌐 Multi-page Streamlit Web Application
